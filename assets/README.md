# 🚀 Stable Diffusion 3 Custom Nodes

## 📌 Project Overview
This project builds advanced custom nodes for Stable Diffusion 3 inside ComfyUI using PyTorch.

## ⚙️ Features
- Triple CLIP Loader
- SD3 Prompt Encoding
- Skip Layer Guidance
- ControlNet integration
