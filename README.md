# Stable Diffusion 3 Custom Nodes (ComfyUI)

## 🚀 Project Overview
Custom nodes for Stable Diffusion 3 inside ComfyUI for advanced image generation control.

---

## ⚙️ Setup Steps

### 1. Clone repository
git clone https://github.com/ammu58/sd3-comfyui-custom-nodes.git

### 2. Enter folder
cd sd3-comfyui-custom-nodes

### 3. Install dependencies
pip install -r requirements.txt

---

## 🧠 Workflow Steps (Colab)

### Step 1: Install libraries
pip install diffusers transformers accelerate

### Step 2: Run model
Use SD3 pipeline to generate images
